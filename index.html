HTML Basics

What is HTML?

HTML or "HyperTextMarkup Language" is used to give content to a webpage and instructs web browsers on how to structure that content.

An HTML element is a piece of content in an HTML document and uses the following syntax: opening tag + content + closing tag. In the code provided.

Common HTML Elements

HTML Element Tag Use
Headings <h1>-<h6> Allows you to specify that certain parts of your content are headings
Paragraph <p> Contains paragraphs of the text.
Lists <ul> Unordered list - the order of the list item or <li> does NOT matter.
<ol> Ordered list - the order of the list item or <li> DOES matter.
Div <div> A container that divides an HTML document into sections. Can contain flow content such as headings, paragraphs, links, images, etc.
Span <span> An inline container for text and can be used to group text for styling purposes.
Images <img> Contains images in the document. The src attribute is mandatory and contains the image URL. Does NOT have a closing tag.
Line Break <br> Will create a line break in the text. Does NOT have a closing tag.
Anchor <a> An anchor element can make the text it encloses into a hyperlink; it consists of a hyperlink and the clickable text. Anchors can take a number of attributes, such as href, target, alt
href or Hypertext Reference indicates a relationship between pages. <a href="www.google.com" target="_blank" alt=Google.com>Google</a>

HTML Attributes

Elements can also have attributes. They contain extra information in the opening tag that won't appear in the contents. It may also configure or change the element's default behavior.

An attribute should have: a space between the element and the name, the attribute name followed by an "=", and an attribute value wrapped in quotes. Always include quotes, as it results in more readable code. Single or double quotes is simply a matter of style.

Different types of attributes include src, class, id, href, target, alt.
